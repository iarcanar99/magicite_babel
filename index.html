🎛️ ปุ่มควบคุมหลัก:

  • • START: เริ่ม-หยุดการแปล (hotkey F9)
  • • TUI: เปิด-ปิดหน้าต่างผลการแปล
  • • LOG: เปิดประวัติการแปลแบบ Chat Bubble
  • • CON: Control UI ศูนย์ควบคุมพื้นที่และ Preset
  • • MINI: สลับเป็นโหมดย่อเพื่อประหยัดพื้นที่

Control UI - ศูนย์ควบคุม

แผงควบคุมที่รวมฟีเจอร์สำคัญทั้งหมด! ออกแบบมาให้ใช้งานง่าย มีทุกสิ่งที่ต้องการในที่เดียว

🎯 ฟังก์ชันหลัก:

  • • ชุดการตั้งค่า (1-6): สลับการตั้งค่าได้รวดเร็ว
  • • พื้นที่แปล A/B/C: เลือกและกำหนดพื้นที่แปล
  • • แปลซ้ำ: บังคับแปลซ้ำทันที
  • • บันทึก/โหลด: จัดเก็บชุดการตั้งค่า
  • • แปลแบบคลิก/ชี้: เปิด-ปิดการแปลแบบ manual

หน้าต่างแสดงผลการแปล

🎨 การแสดงผล

สวยงามและชัดเจน เชื่อมโยงข้อมูลกับ NPC Manager หากพบตัวละครที่มีในระบบจะแสดง เครื่องหมายถูกสีเขียว และสามารถคลิกที่ชื่อตัวละครเพื่อเพิ่มข้อมูลได้ทันที

  • • เปลี่ยนฟอนต์: ปรับฟอนต์ได้ที่ Settings > Font
  • • เลือกสีพื้นหลัง: ปรับสีและความโปร่งใส

🔒 ระบบล็อก 3 แบบ

ปกติ, ล็อก+ซ่อนพื้นหลัง (เสพกราฟิกเต็มตา), และล็อก+มีพื้นหลัง (อ่านง่าย ไม่โดนมือ)

⚡ ปุ่มแปลซ้ำ:

  • • ปุ่ม Force: คลิกเพื่อเปิดปุ่มสำหรับการแตะเพื่อแปลซ้ำ
  • • คลิกขวา: บนพื้นหลังตำแหน่งใดก็ได้

LOG - หน้าต่างประวัติการแปล

ไม่พลาดทุกบทสนทนา! หน้าต่างนี้จะบันทึก การแปลทั้งหมด ตั้งแต่เริ่มเปิดโปรแกรม ในรูปแบบ แชทบับเบิล สมัยใหม่ ช่วยให้ย้อนกลับไปอ่านเนื้อเรื่องหรือบทพูดที่อาจจะพลาดไปได้ง่ายๆ

✨ ใหม่ในเวอร์ชั่น 9.1:

  • • ดีไซน์ใหม่: แบบบับเบิลดูทันสมัยขึ้น
  • • ล็อกตำแหน่งได้: คงตำแหน่งหน้าต่างไว้
  • • ลดข้อความซ้ำ: ระบบอัจฉริยะไม่แสดงข้อความซ้ำ
  • • Font Manager: ปุ่ม Font Manager เรียกโมดูลได้จาก UI นี้ด้วย
  • • สลับการเรียงข้อความ: บนลง-ล่างขึ้นบน ได้อิสระ

💡 รองรับรูปแบบข้อความทุกประเภท: บทสนทนาปกติ, ข้อความระบบ, ตัวเลือกคำตอบ และผู้พูดที่ไม่รู้จัก

🧠 NPC Manager - ระบบแปลอัจฉริยะ

สุดยอดการแปลที่เป็นธรรมชาติให้ทุกตัวละครมีชีวิตและมีมิติ ทำให้ AI รู้จักและจดจำตัวละครในเกม สร้างประสบการณ์การแปลที่เหมือนซับไตเติ้ลคุณภาพสูง

📋 ข้อมูลตัวละคร:

  • • ชื่อและนามสกุล: บันทึกชื่อเต็มและชื่อเล่น
  • • เพศและสรรพนาม: กำหนดเพศและคำสรรพนาม
  • • บุคลิกการพูด: สุภาพ, ห้วน, ฉลาด, เจ้าเล่ห์, อื่นๆ
  • • Role ในเกม: หลัก, รอง, ประกอบ, ศัตรู

🎯 Enhanced Speaker Detection:

ระบบตรวจสอบเพื่อเพิ่มความแม่นยำในการรับบทสนทนาของตัวละคร ช่วยให้จดจำความมีชื่อผู้พูดได้ทุกๆครั้ง ต้องไม่มีการหลุดแปลฟองคำพูดที่ไม่รู้จัก

🎭 การแปลตามบุคลิกที่กำหนด:

Magicite Babel v9.1 - อัปเดตครั้งใหญ่

🚀 การปรับปรุงครั้งใหญ่ใน v9.1

⚡ ประสิทธิภาพที่เหนือชั้น

  • 40-50% CPU Usage Reduction: ลดการใช้ CPU อย่างมาก ทำให้เกมเล่นได้ลื่นไหลขึ้น
  • OCR Pipeline Optimization: กำจัดการประมวลผลที่ใช้ CPU สูง เช่น CLAHE และ fastNlMeansDenoising
  • Enhanced Screen Capture: ระบบจับภาพหน้าจอแบบ single capture ลดการใช้ I/O ถึง 85%
  • Rapid Text Detection: ตรวจจับข้อความที่เปลี่ยนภายใน 1 วินาทีได้อย่างแม่นยำ

🧠 ปรับปรุงคุณภาพการแปล

  • FFXIV Context Integration: การแปล Choice ที่แม่นยำขึ้น เช่น "reflections" → "โลกสะท้อน"
  • Warrior of Light Voice: น้ำเสียงตัวเอกที่สอดคล้อง (กล้าหาญ หนักแน่น มั่นใจ)
  • Enhanced Translation Caching: ระบบแคชที่เข้าใจบริบทและตัวละคร ลด API calls ถึง 40-60%
  • Context-Aware Translation: การแปลที่คำนึงถึงบริบทของสถานการณ์และตัวละคร

🛠️ ปรับปรุงระบบและเสถียรภาพ

  • Thread Conflict Resolution: แก้ไขปัญหา Splash Screen Thread ที่ทำให้โปรแกรมค้าง
  • Intelligent Cache Management: ระบบจัดการแคชอัจฉริยะพร้อม TTL และ LRU eviction
  • Memory Management: การจัดการหน่วยความจำที่มีประสิทธิภาพขึ้น
  • Error Recovery System: ระบบกู้คืนจากข้อผิดพลาดอัตโนมัติ

📈 ผลลัพธ์ที่วัดได้

ก่อนการอัปเดต v9.1:

  • • OCR processing ใช้ CPU สูง
  • • หลาย screen captures ต่อรอบ
  • • ไม่มีระบบแคช
  • • การตรวจจับข้อความช้าในโหมด rapid

หลังการอัปเดต v9.1:

  • • OCR processing ที่เบาขึ้น (PIL-only)
  • • Single screen capture + area cropping
  • • Intelligent caching system
  • • Sub-second detection สำหรับ rapid dialogue
