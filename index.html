สัมผัสประสบการณ์การเล่นเกมระดับใหม่ด้วย AI แปลภาษาคุณภาพสูงราวกับดูซีรีส์เน็ทฟลิกซ์ที่มีซับไตเติลคุณภาพเยี่ยม ที่เข้าใจบริบทและคาแรกเตอร์ของตัวละครได้อย่างลึกซึ้ง ไม่ใช่แค่แปลคำต่อคำ แต่เข้าใจคาแรกเตอร์ตัวละครด้วยระบบ NPC Manager ล้ำสมัย

Magicite Babel สร้างโดยคนไทยผู้หลงใหลในเกมและซีรีส์ Final Fantasy โดยมีจุดประสงค์เพื่อทลายกำแพงภาษาให้ผู้เล่นชาวไทยได้ดื่มด่ำกับเนื้อเรื่องอันลึกซึ้ง โดยเฉพาะอย่างยิ่งในเกม FFXIV ที่ใช้ภาษาโบราณและซับซ้อน แม้จะมีความรู้ภาษาอังกฤษพื้นฐาน ก็ยังไม่เพียงพอที่จะเข้าใจและซึมซับอรรถรสทั้งหมดได้อย่างสมบูรณ์

รองรับโมเดล AI ชั้นนำทั้ง Gemini, GPT และ Claude พร้อมฟีเจอร์ Hover Translation และ Preset อัจฉริยะที่ช่วยให้การแปลราบรื่น รวดเร็ว อ่านบทสนทนาที่เคยสับสนด้วยการแปลที่เป็นธรรมชาติเหมือนคนไทยเขียนเอง

ใช้ OCR ดึงข้อความจากหน้าจอเกมและแปลทันทีด้วย AI ขั้นสูง รองรับหลากหลายโมเดล (Gemini, GPT, Claude)

Hover Translation

แปลเมื่อเมาส์ชี้เท่านั้น ไม่ต้องคลิกให้เสียเวลา สะดวกยิ่งขึ้นกับการอ่านข้อความในเกม

ระบบจัดการ NPC

ปรับแต่งน้ำเสียงและลักษณะการพูดของตัวละครให้เป็นธรรมชาติ เก็บข้อมูลและบุคลิกตัวละครไว้ใช้งานต่อเนื่อง

ปรับแต่ง UI ในแบบที่ชอบ

เลือกธีม ฟอนต์ ขนาด และตำแหน่งได้ตามต้องการ ปรับความโปร่งใสของพื้นหลังให้เข้ากับเกมของคุณ

ระบบ Preset อัจฉริยะ

บันทึกพื้นที่แปลและการตั้งค่าเป็นชุด (Preset) เพื่อสลับไปมาได้อย่างง่ายดายและรวดเร็ว

Toggle Switches

ปุ่มเปิด/ปิดฟีเจอร์ต่างๆ ได้อย่างรวดเร็ว ปรับการใช้งานได้ตามความต้องการในแต่ละช่วงเวลา

คู่มือการใช้งาน

1 เลือกพื้นที่สำหรับการแปล (Area A+B)

เลือกพื้นที่สำหรับการแปลโดยกำหนด Area-A สำหรับส่วนชื่อตัวละคร และ Area-B สำหรับส่วนข้อความสนทนา ควรเลือกทั้ง A และ B ร่วมกันเพื่อให้ระบบสามารถแยกชื่อและข้อความได้อย่างถูกต้อง หลังจากเลือกพื้นที่แล้ว ใช้ Show Area เพื่อตรวจสอบกรอบพื้นที่ที่เลือก

2 เริ่มต้นการแปลและการตั้งค่า Preset

กดปุ่ม Start Translation เพื่อเริ่มกระบวนการแปลอัตโนมัติ ระบบจะตรวจจับข้อความในพื้นที่ที่กำหนดและส่งไปแปลทันที หากต้องการหยุดการแปลชั่วคราว สามารถกดปุ่ม hotkey (ค่าเริ่มต้นคือ W A S D) ที่ตั้งไว้ในหน้าตั้งค่า

ใช้ Preset (1-6) บน Control UI เพื่อบันทึกและสลับระหว่างชุดพื้นที่แปลต่างๆ ได้อย่างรวดเร็ว โดย Preset 1-3 สำหรับการใช้งานหลัก และ Preset 4-6 สำหรับการตั้งค่าแบบกำหนดเอง

3 การแสดงผลและฟีเจอร์เสริม

ผลการแปลจะแสดงในหน้าต่าง TUI (Translation UI) ซึ่งสามารถเปิด-ปิดได้ด้วยปุ่ม TUI ในหน้าต่าง Control UI

ปรับโหมดการแสดงผลได้หลายรูปแบบ เช่น พื้นหลัง+ล็อคตำแหน่ง, ซ่อนพื้นหลัง+ล็อคตำแหน่ง หรือพื้นหลัง+ปลดล็อก เพื่อปรับให้เหมาะกับรูปแบบการเล่นเกม

หากต้องการบังคับแปลซ้ำ สามารถ แตะปุ่ม F หรือ คลิกขวา ที่พื้นหลังของหน้าต่างแปล หรือกดปุ่ม Force บนหน้าต่าง Control UI ได้ทันที ดูประวัติการแปลได้ที่ปุ่ม LOG

Magicite Babel ทำงานอย่างปลอดภัยและสอดคล้องตามกฎของเกม

โปรแกรม Magicite Babel ไม่ใช่ plug-in หรือ mod ของเกม แต่อย่างใด โปรแกรมนี้ทำงานแยกต่างหากจากตัวเกมโดยสมบูรณ์ ด้วยการทำงานในลักษณะ "อ่านภาพหน้าจอ" จากนั้นประมวลผลและแสดงผลแยกต่างหากจากตัวเกม

กระบวนการทำงานคือ:

  1. ถ่ายภาพหน้าจอในบริเวณที่ผู้ใช้กำหนด (ไม่มีการแทรกแซงไฟล์ใดๆ ของเกม)
  2. ส่งภาพไปประมวลผล OCR เพื่อแปลงเป็นข้อความ
  3. ส่งข้อความไปแปลด้วย AI ภายนอก
  4. แสดงผลในหน้าต่างแยกต่างหาก

การทำงานในลักษณะนี้ ไม่มีความเสี่ยงในการละเมิดกฎของเกม เนื่องจากเป็นเพียงการอ่านภาพหน้าจอและแสดงผลแยกต่างหาก ไม่มีการดัดแปลงไฟล์เกม ไม่มีการเข้าถึงข้อมูลภายในเกม และไม่ได้ให้ความได้เปรียบในการเล่นเกมแต่อย่างใด

  • แปลสดจากเกม เรียลไทม์: อ่านข้อความบนหน้าจอเกมแล้วแปลให้ทันทีอย่างลื่นไหล เหมือนมีล่ามส่วนตัวข้างๆ คุณ!
  • เลือก AI Model ชั้นนำได้: ไม่ว่าจะเป็น Gemini, GPT, หรือ Claude คุณเลือกใช้ AI ที่คิดว่าแปลได้ดีที่สุดสำหรับเกมนั้นๆ ได้เลย ให้พลังและความยืดหยุ่นสูงสุด
  • หน้าต่างแปล (TUI) ปรับได้ดั่งใจ: ไม่ใช่แค่ปรับขนาดหรือความใส แต่คุณกำหนดตำแหน่งและหน้าตาของหน้าต่างแปลได้อย่างอิสระเต็มที่ ให้เข้ากับสไตล์การเล่นและไม่บดบังส่วนสำคัญของเกม
  • Preset สุดฉลาด จัดการทุกสถานการณ์: เบื่อไหมกับการต้องมานั่งเลือกพื้นที่แปลซ้ำๆ? ระบบ Preset ช่วยให้คุณบันทึกตำแหน่งที่แปลบ่อยๆ (บทพูด, เนื้อเรื่องในตำรา, ตัวเลือกตอบคำถาม) แล้วสลับใช้ได้ง่ายๆ ทั้งแบบกดเอง หรือให้โปรแกรมสลับอัตโนมัติเมื่อตรวจเจอรูปแบบที่คุ้นเคย จัดการกับ UI เกมที่ซับซ้อนได้สบาย
  • แผงควบคุม (Control Panel) ที่เข้าถึงง่าย: ควบคุมทุกอย่างได้จากที่เดียว ไม่ว่าจะเป็นการเลือก Preset, กดบังคับแปลเมื่อต้องการ, หรือแม้แต่จำกัดการทำงาน CPU เพื่อไม่ให้กระทบประสิทธิภาพเกม
  • แต่งสวยด้วย Theme: ปรับแต่งสีสันและหน้าตาโปรแกรมให้เป็นสไตล์ของคุณเอง สร้างบรรยากาศที่ชอบได้เลย
  • ✨ รู้จักตัวละคร (NPC Manager) - ยกระดับการแปลเทียบชั้นซับไตเติ้ลมืออาชีพ! ✨: นี่คือฟีเจอร์ที่ทำให้ MBB โดดเด่น! คุณสามารถป้อนข้อมูลตัวละคร (ชื่อ, เพศ, สรรพนามที่ควรใช้) ให้ AI เรียนรู้ ทำให้การแปลชื่อและคำพูดติดปากมีความสม่ำเสมอ ถูกต้องตามบริบทและลักษณะนิสัยของตัวละคร หมดปัญหาการแปลชื่อผิดๆ ถูกๆ หรือใช้สรรพนามสลับไปมา ผลลัพธ์ที่ได้คือคำแปลคุณภาพสูง มีอรรถรส เหมือนกำลังดูซีรีส์ดีๆ พร้อมซับไทยคุณภาพเยี่ยม!
  • Fade Out อัตโนมัติ ไม่เกะกะ: เมื่อไม่มีข้อความใหม่ หน้าต่างแปลจะค่อยๆ จางหายไปเอง ไม่บดบังทัศนียภาพในเกม และจะกลับมาเมื่อมีข้อความใหม่
  • โหมดบังคับแปล (Manual Force): ควบคุมได้เต็มที่! เลือกเปิด/ปิดการแปลอัตโนมัติได้ตามใจชอบ เหมาะสำหรับฉากที่ไม่ต้องการให้แปลตลอดเวลา
  • ชี้แล้วแปล (Hover-to-Force): สะดวกสุดๆ แค่เอาเมาส์ไปชี้ค้างไว้ที่ปุ่ม "Force" ก็สั่งให้โปรแกรมเริ่มแปลได้ทันที ไม่ต้องคลิกให้เสียจังหวะ
