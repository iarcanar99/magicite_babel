Magicite Babel

แปลภาษาเกมแบบเรียลไทม์ ด้วยเทคโนโลยี AI

สัมผัสประสบการณ์การเล่นเกมระดับใหม่ด้วย AI แปลภาษาคุณภาพสูงราวกับดูซีรีส์เน็ทฟลิกซ์ที่มีซับไตเติลคุณภาพเยี่ยม ที่เข้าใจบริบทและคาแรกเตอร์ของตัวละครได้อย่างลึกซึ้ง ไม่ใช่แค่แปลคำต่อคำ แต่เข้าใจคาแรกเตอร์ตัวละครด้วยระบบ NPC Manager ล้ำสมัย

Magicite Babel สร้างโดยคนไทยผู้หลงใหลในเกมและซีรีส์ Final Fantasy โดยมีจุดประสงค์เพื่อทลายกำแพงภาษาให้ผู้เล่นชาวไทยได้ดื่มด่ำกับเนื้อเรื่องอันลึกซึ้ง โดยเฉพาะอย่างยิ่งในเกม FFXIV ที่ใช้ภาษาโบราณและซับซ้อน แม้จะมีความรู้ภาษาอังกฤษพื้นฐาน ก็ยังไม่เพียงพอที่จะเข้าใจและซึมซับอรรถรสทั้งหมดได้อย่างสมบูรณ์

รองรับโมเดล AI ชั้นนำทั้ง Gemini, GPT และ Claude พร้อมฟีเจอร์ Hover Translation และ Preset อัจฉริยะที่ช่วยให้การแปลราบรื่น รวดเร็ว อ่านบทสนทนาที่เคยสับสนด้วยการแปลที่เป็นธรรมชาติเหมือนคนไทยเขียนเอง

ดูฟีเจอร์ ดาวน์โหลดเลย

ฟีเจอร์เด่น

แปลภาษาแบบเรียลไทม์

ใช้ OCR ดึงข้อความจากหน้าจอเกมและแปลทันทีด้วย AI ขั้นสูง รองรับหลากหลายโมเดล (Gemini, GPT, Claude)

Hover Translation

แปลเมื่อเมาส์ชี้เท่านั้น ไม่ต้องคลิกให้เสียเวลา สะดวกยิ่งขึ้นกับการอ่านข้อความในเกม

ระบบจัดการ NPC

ปรับแต่งน้ำเสียงและลักษณะการพูดของตัวละครให้เป็นธรรมชาติ เก็บข้อมูลและบุคลิกตัวละครไว้ใช้งานต่อเนื่อง

ปรับแต่ง UI ในแบบที่ชอบ

เลือกธีม ฟอนต์ ขนาด และตำแหน่งได้ตามต้องการ ปรับความโปร่งใสของพื้นหลังให้เข้ากับเกมของคุณ

ระบบ Preset อัจฉริยะ

บันทึกพื้นที่แปลและการตั้งค่าเป็นชุด (Preset) เพื่อสลับไปมาได้อย่างง่ายดายและรวดเร็ว

Toggle Switches

ปุ่มเปิด/ปิดฟีเจอร์ต่างๆ ได้อย่างรวดเร็ว ปรับการใช้งานได้ตามความต้องการในแต่ละช่วงเวลา

คู่มือการใช้งาน

1 เลือกพื้นที่ที่ต้องการแปล

เลือกพื้นที่ที่ต้องการให้โปรแกรมแปลได้แยกตามประเภท เช่น A สำหรับส่วนชื่อตัวละคร และ B สำหรับส่วนข้อความสนทนา จากนั้นตรวจสอบพื้นที่ที่เลือกด้วยการกด Show Area เพื่อดูกรอบสีแดงแสดงพื้นที่ที่เลือก

2 เริ่มการแปล

กดปุ่ม Start Translation เพื่อเริ่มให้โปรแกรมแปลภาษาโดยอัตโนมัติ เมื่อตรวจพบข้อความในพื้นที่ที่เลือกไว้ โปรแกรมจะใช้ OCR อ่านข้อความและส่งไปให้ AI แปล

คุณสามารถกดปุ่ม W A S D เพื่อหยุดการแปลชั่วคราวได้ (สามารถตั้งค่าได้ในหน้าตั้งค่า)

3 ดูผลการแปล

ผลการแปลจะแสดงในหน้าต่าง TUI (Translation UI) ซึ่งคุณสามารถเปิด-ปิดได้ด้วยปุ่ม TUI

คุณสามารถเลือกโหมดต่างๆ เช่น พื้นหลัง+ล็อคตำแหน่ง, ซ่อนพื้นหลัง+ล็อค, หรือพื้นหลัง+ปลดล็อก เพื่อปรับการแสดงผลให้เหมาะกับรูปแบบการเล่นเกมของคุณ

กดปุ่ม F หรือคลิกที่พื้นหลังของหน้าต่างแปลเพื่อบังคับแปลซ้ำได้ทันที

หน้าตาโปรแกรม

หน้าต่างหลัก

ศูนย์กลางควบคุม Magicite Babel จัดการทุกอย่างตั้งแต่เลือกพื้นที่แปล, เริ่ม/หยุดการแปลอัตโนมัติ, เปิด/ปิด Hover Translation, เข้าถึง Settings และ Control UI

ผลลัพธ์การแปล (TUI)

หน้าต่างแสดงผลการแปลจาก AI ที่อ่านง่าย ชัดเจน พร้อมระบุชื่อตัวละคร (ถ้ามี) ด้วยความฉลาดของ NPC Manager ที่ช่วยให้การแปลสม่ำเสมอและตรงคาแรกเตอร์

✨ พิเศษ! ปรับแต่งฟอนต์ได้เต็มที่ ทั้งหน้าตาและขนาด ผ่านฟีเจอร์ 'จัดการฟอนต์' ในหน้า Settings > FONT ให้เข้ากับสไตล์ของคุณที่สุด

Control UI

แผงควบคุมย่อยสุดเทพ! สลับ Preset (1-6) ได้รวดเร็ว, เลือกพื้นที่แปล (A/B/C) สำหรับ Custom Preset (4-6), กด Force แปลซ้ำ, เปิด/ปิดโหมดคลิก/ชี้แปล และดูสถานะ CPU

✨ พิเศษ! มี Hover-to-Select Preset (1-3) และ Hover-to-Force ด้วยนะ!

ประวัติการแปล

ไม่พลาดทุกบทสนทนา! หน้าต่างนี้จะบันทึก Log การแปลทั้งหมด ตั้งแต่เริ่มเปิดโปรแกรม ช่วยให้ย้อนกลับไปอ่านเนื้อเรื่องหรือบทพูดที่อาจจะพลาดไปได้ง่ายๆ

NPC Manager

ฟีเจอร์สุดล้ำ! ช่วยให้ AI รู้จักและจดจำตัวละครในเกม กำหนดชื่อ, เพศ, สรรพนาม และลักษณะการพูดเฉพาะตัว ทำให้การแปลชื่อและคำพูด ถูกต้อง แม่นยำ และมีชีวิตชีวา เหมือนมีคนแปลจริงๆ!

การตั้งค่า

ปรับแต่ง Magicite Babel ได้ทุกอย่าง! ตั้งค่า API Keys (Gemini/GPT/Claude), เลือก Model, ปรับ Prompt, ตั้งค่า OCR, กำหนดปุ่ม Hotkeys, จัดการ Theme & Font และอื่นๆ อีกเพียบ

ปรับแต่งธีม

เปลี่ยนโฉมโปรแกรมให้เข้ากับสไตล์ของคุณ! เลือก Theme ที่มีให้ หรือ สร้าง Theme ใหม่ เอง ปรับแต่งสีสันต่างๆ ของ UI ได้ตามใจชอบผ่าน AppearanceManager

จัดการฟอนต์

นอกจากการปรับธีมแล้ว คุณยังปรับแต่ง ฟอนต์ ที่ใช้แสดงในหน้าต่างแปล (TUI) ได้เต็มที่! เลือกฟอนต์สวยๆ ที่มีในเครื่อง ปรับขนาด และสไตล์ให้อ่านง่าย สบายตา ผ่านหน้าต่าง 'จัดการฟอนต์' (FontUI) โดยเฉพาะ

โหมดมินิ

อยากได้ UI เล็กๆ ไม่เกะกะจอ? Mini Mode ช่วยได้! ย่อหน้าต่างควบคุมให้เหลือแต่ปุ่มที่จำเป็น เช่น สลับ Preset, Force แปล และแสดงสถานะพื้นฐาน

Magicite Babel - Patch Notes 🚀

✨ ฟีเจอร์เด่นของ Magicite Babel ✨

เราอยากทบทวนฟีเจอร์เจ๋งๆ ที่ทำให้ Magicite Babel แตกต่างและมอบประสบการณ์ที่ดีที่สุดให้คุณ:

  • แปลสดจากเกม เรียลไทม์: อ่านข้อความบนหน้าจอเกมแล้วแปลให้ทันทีอย่างลื่นไหล เหมือนมีล่ามส่วนตัวข้างๆ คุณ!
  • เลือก AI Model ชั้นนำได้: ไม่ว่าจะเป็น Gemini, GPT, หรือ Claude คุณเลือกใช้ AI ที่คิดว่าแปลได้ดีที่สุดสำหรับเกมนั้นๆ ได้เลย ให้พลังและความยืดหยุ่นสูงสุด
  • หน้าต่างแปล (TUI) ปรับได้ดั่งใจ: ไม่ใช่แค่ปรับขนาดหรือความใส แต่คุณกำหนดตำแหน่งและหน้าตาของหน้าต่างแปลได้อย่างอิสระเต็มที่ ให้เข้ากับสไตล์การเล่นและไม่บดบังส่วนสำคัญของเกม
  • Preset สุดฉลาด จัดการทุกสถานการณ์: เบื่อไหมกับการต้องมานั่งเลือกพื้นที่แปลซ้ำๆ? ระบบ Preset ช่วยให้คุณบันทึกตำแหน่งที่แปลบ่อยๆ (บทพูด, เนื้อเรื่องในตำรา, ตัวเลือกตอบคำถาม) แล้วสลับใช้ได้ง่ายๆ ทั้งแบบกดเอง หรือให้โปรแกรมสลับอัตโนมัติเมื่อตรวจเจอรูปแบบที่คุ้นเคย จัดการกับ UI เกมที่ซับซ้อนได้สบาย
  • แผงควบคุม (Control Panel) ที่เข้าถึงง่าย: ควบคุมทุกอย่างได้จากที่เดียว ไม่ว่าจะเป็นการเลือก Preset, กดบังคับแปลเมื่อต้องการ, หรือแม้แต่จำกัดการทำงาน CPU เพื่อไม่ให้กระทบประสิทธิภาพเกม
  • แต่งสวยด้วย Theme: ปรับแต่งสีสันและหน้าตาโปรแกรมให้เป็นสไตล์ของคุณเอง สร้างบรรยากาศที่ชอบได้เลย
  • ✨ รู้จักตัวละคร (NPC Manager) - ยกระดับการแปลเทียบชั้นซับไตเติ้ลมืออาชีพ! ✨: นี่คือฟีเจอร์ที่ทำให้ MBB โดดเด่น! คุณสามารถป้อนข้อมูลตัวละคร (ชื่อ, เพศ, สรรพนามที่ควรใช้) ให้ AI เรียนรู้ ทำให้การแปลชื่อและคำพูดติดปากมีความสม่ำเสมอ ถูกต้องตามบริบทและลักษณะนิสัยของตัวละคร หมดปัญหาการแปลชื่อผิดๆ ถูกๆ หรือใช้สรรพนามสลับไปมา ผลลัพธ์ที่ได้คือคำแปลคุณภาพสูง มีอรรถรส เหมือนกำลังดูซีรีส์ดีๆ พร้อมซับไทยคุณภาพเยี่ยม!
  • Fade Out อัตโนมัติ ไม่เกะกะ: เมื่อไม่มีข้อความใหม่ หน้าต่างแปลจะค่อยๆ จางหายไปเอง ไม่บดบังทัศนียภาพในเกม และจะกลับมาเมื่อมีข้อความใหม่
  • โหมดบังคับแปล (Manual Force): ควบคุมได้เต็มที่! เลือกเปิด/ปิดการแปลอัตโนมัติได้ตามใจชอบ เหมาะสำหรับฉากที่ไม่ต้องการให้แปลตลอดเวลา
  • ชี้แล้วแปล (Hover-to-Force): สะดวกสุดๆ แค่เอาเมาส์ไปชี้ค้างไว้ที่ปุ่ม "Force" ก็สั่งให้โปรแกรมเริ่มแปลได้ทันที ไม่ต้องคลิกให้เสียจังหวะ

  • ✨ ใหม่! Hover Translation (ชี้เพื่อสลับ Preset): แค่เอาเมาส์ไปชี้ที่พื้นที่ของ Preset หลัก (1-3) โปรแกรมก็จะสลับไปใช้ Preset นั้นให้เลย! สะดวกสุดๆ เวลาเจอหน้าจอที่มีหลายส่วน (เราปรับปรุงให้ฉลาดขึ้นด้วยนะ จะได้ไม่สลับมั่วตอนเผลอเลื่อนเมาส์ผ่านเร็วๆ หรือตอนยังแก้ค่าไม่เสร็จ 😉)

16 เมษายน 2025

  • ✨ Gemini 2.0 Flash มาแล้ว!: เพิ่มพลังแปลด้วยโมเดลใหม่ล่าสุด Gemini-2.0-Flash เร็วขึ้น คุณภาพดีขึ้นกว่าเดิม!
  • ✨ Hover-to-Force ทำงานได้ทุกปุ่ม: ฟีเจอร์ชี้ค้างเพื่อสั่งแปล ตอนนี้ใช้ได้กับทุกปุ่มที่เกี่ยวกับการ Force Translate เลยนะ!
  • ปรับปรุง Control Panel: หน้าตาดูดีขึ้น ปุ่ม Preset กับส่วนควบคุม CPU ใช้งานง่ายกว่าเดิม
  • แก้บั๊กบันทึก Preset: ปัญหาตั้งชื่อ Custom Preset แล้วบันทึกไม่ได้ ถูกแก้ไขแล้ว!
  • ปรับปรุงหน้าตั้งค่า Model: หน้าตา UI/UX ดีขึ้น ปุ่ม Apply ชัดเจน จะกดได้ก็ต่อเมื่อมีการแก้ค่าเท่านั้น

หวังว่าทุกคนจะชอบอัปเดตเหล่านี้นะคะ! ถ้ามีข้อเสนอแนะหรือเจอปัญหาอะไร แจ้งทีมงานได้เลยค่ะ 😊
