แปลภาษาเกมแบบเรียลไทม์ ด้วยเทคโนโลยี AI

สัมผัสประสบการณ์การเล่นเกมระดับใหม่ด้วย AI แปลภาษาคุณภาพสูงราวกับดูซีรีส์เน็ทฟลิกซ์ที่มีซับไตเติลคุณภาพเยี่ยม ที่เข้าใจบริบทและคาแรกเตอร์ของตัวละครได้อย่างลึกซึ้ง ไม่ใช่แค่แปลคำต่อคำ แต่เข้าใจคาแรกเตอร์ตัวละครด้วยระบบ NPC Manager ล้ำสมัย

Magicite Babel สร้างโดยคนไทยผู้หลงใหลในเกมและซีรีส์ Final Fantasy โดยมีจุดประสงค์เพื่อทลายกำแพงภาษาให้ผู้เล่นชาวไทยได้ดื่มด่ำกับเนื้อเรื่องอันลึกซึ้ง โดยเฉพาะอย่างยิ่งในเกม FFXIV ที่ใช้ภาษาโบราณและซับซ้อน แม้จะมีความรู้ภาษาอังกฤษพื้นฐาน ก็ยังไม่เพียงพอที่จะเข้าใจและซึมซับอรรถรสทั้งหมดได้อย่างสมบูรณ์

รองรับโมเดล AI ชั้นนำทั้ง Gemini, GPT และ Claude พร้อมฟีเจอร์ Hover Translation และ Preset อัจฉริยะที่ช่วยให้การแปลราบรื่น รวดเร็ว อ่านบทสนทนาที่เคยสับสนด้วยการแปลที่เป็นธรรมชาติเหมือนคนไทยเขียนเอง

ฟีเจอร์เด่น

แปลภาษาแบบเรียลไทม์

ใช้ OCR ดึงข้อความจากหน้าจอเกมและแปลทันทีด้วย AI ขั้นสูง รองรับหลากหลายโมเดล (Gemini, GPT, Claude)

Hover Translation

แปลเมื่อเมาส์ชี้เท่านั้น ไม่ต้องคลิกให้เสียเวลา สะดวกยิ่งขึ้นกับการอ่านข้อความในเกม

ระบบจัดการ NPC

ปรับแต่งน้ำเสียงและลักษณะการพูดของตัวละครให้เป็นธรรมชาติ เก็บข้อมูลและบุคลิกตัวละครไว้ใช้งานต่อเนื่อง

ปรับแต่ง UI สูง

เลือกธีม ฟอนต์ ขนาด และตำแหน่งได้ตามต้องการ ปรับความโปร่งใสของพื้นหลังให้เข้ากับเกมของคุณ

ระบบ Preset อัจฉริยะ

บันทึกพื้นที่แปลและการตั้งค่าเป็นชุด (Preset) เพื่อสลับไปมาได้อย่างง่ายดายและรวดเร็ว

Toggle Switches

ปุ่มเปิด/ปิดฟีเจอร์ต่างๆ ได้อย่างรวดเร็ว ปรับการใช้งานได้ตามความต้องการในแต่ละช่วงเวลา

การใช้งานเบื้องต้น

1 เลือกพื้นที่ที่ต้องการแปล

เลือกพื้นที่ที่ต้องการให้โปรแกรมแปลได้แยกตามประเภท เช่น A สำหรับส่วนชื่อตัวละคร และ B สำหรับส่วนข้อความสนทนา จากนั้นตรวจสอบพื้นที่ที่เลือกด้วยการกด Show Area เพื่อดูกรอบสีแดงแสดงพื้นที่ที่เลือก

2 เริ่มการแปล

กดปุ่ม Start Translation เพื่อเริ่มให้โปรแกรมแปลภาษาโดยอัตโนมัติ เมื่อตรวจพบข้อความในพื้นที่ที่เลือกไว้ โปรแกรมจะใช้ OCR อ่านข้อความและส่งไปให้ AI แปล

คุณสามารถกดปุ่ม W A S D เพื่อหยุดการแปลชั่วคราวได้ (สามารถตั้งค่าได้ในหน้าตั้งค่า)

3 ดูผลการแปล

ผลการแปลจะแสดงในหน้าต่าง TUI (Translation UI) ซึ่งคุณสามารถเปิด-ปิดได้ด้วยปุ่ม TUI

คุณสามารถเลือกโหมดต่างๆ เช่น พื้นหลัง+ล็อคตำแหน่ง, ซ่อนพื้นหลัง+ล็อค, หรือพื้นหลัง+ปลดล็อก เพื่อปรับการแสดงผลให้เหมาะกับรูปแบบการเล่นเกมของคุณ

กดปุ่ม F หรือคลิกที่พื้นหลังของหน้าต่างแปลเพื่อบังคับแปลซ้ำได้ทันที

หน้าตาโปรแกรม

หน้าต่างหลัก

ควบคุมการทำงานพื้นฐาน เลือกพื้นที่แปล เริ่ม/หยุดการแปล และเข้าถึงการตั้งค่าต่างๆ ดีไซน์ที่เรียบง่ายแต่ครบครัน ให้คุณเข้าถึงฟีเจอร์ต่างๆ ได้อย่างรวดเร็ว

ผลลัพธ์การแปล

แสดงผลการแปลจาก AI พร้อมระบุชื่อตัวละครและข้อความที่แปลแล้วอย่างชัดเจน ด้วยระบบ NPC Manager ที่ทำให้เข้าใจคาแรกเตอร์ทำให้การแปลมีความสม่ำเสมอและถูกต้องตามบริบท

Control UI

ควบคุมการสลับพื้นที่และ preset ต่างๆ พร้อมปุ่ม Force เพื่อแปลซ้ำเมื่อต้องการ ใช้งานง่ายด้วยฟีเจอร์ Hover เพียงชี้เมาส์ที่พื้นที่ Preset (1-3) โปรแกรมจะสลับให้อัตโนมัติ

ประวัติการแปล

บันทึกทุกบทสนทนาที่แปลแล้วตั้งแต่เริ่มต้นใช้งาน ช่วยให้ย้อนกลับไปดูบทสนทนาก่อนหน้าได้ สะดวกสำหรับการติดตามเนื้อเรื่องที่ซับซ้อน โดยเฉพาะในเกม RPG ที่มีเนื้อเรื่องยาว

NPC Manager

จัดการข้อมูลและบุคลิกของตัวละครเพื่อให้การแปลเป็นธรรมชาติยิ่งขึ้น กำหนดสไตล์การแปลเฉพาะตัวละคร นำเสนอคาแรกเตอร์ที่สอดคล้องกับต้นฉบับ ทำให้การแปลราวกับดูซีรีส์ที่มีซับไตเติลคุณภาพเยี่ยม

การตั้งค่า

ปรับแต่งพารามิเตอร์ของ AI, OCR, การแสดงผล และการทำงานของโปรแกรมอย่างละเอียด ตั้งค่า API Key จากผู้ให้บริการ AI ต่างๆ ได้ (แนะนำ Gemini เพราะเร็วและฟรี) ปรับแต่ง prompt เพื่อให้ได้ผลลัพธ์ที่ดีที่สุดสำหรับเกมแต่ละประเภท

ปรับแต่งธีม

เลือกสีและรูปแบบของ UI ให้เข้ากับธีมของเกมและความชอบส่วนตัว ปรับแต่งสีได้ตามต้องการ เพิ่มความโปร่งใสของพื้นหลัง และอื่นๆ อีกมากมาย มีธีมให้เลือกหลากหลายหรือสร้างธีมใหม่เองได้

จัดการฟอนต์

เลือกฟอนต์และปรับขนาดตัวอักษรสำหรับการแสดงผลที่เหมาะสมที่สุด นำเข้าฟอนต์เพิ่มเติมได้อย่างง่ายดาย มาพร้อมกับฟอนต์ไทยยอดนิยมหลากหลายให้เลือกใช้ได้ทันที รองรับฟอนต์ต่างประเทศด้วย

โหมดมินิ

ใช้งานในรูปแบบย่อเพื่อให้รบกวนหน้าจอเกมน้อยที่สุด สลับกลับไปโหมดเต็มได้ง่าย พร้อมปุ่มควบคุมที่จำเป็นในรูปแบบกะทัดรัด ไม่เกะกะหน้าจอแต่ยังคงฟังก์ชันสำคัญไว้ครบถ้วน

Hover Translation (20 เมษายน)

แค่เอาเมาส์ไปชี้ที่พื้นที่ของ Preset หลัก (1-3) โปรแกรมก็จะสลับไปใช้ Preset นั้นให้เลย! สะดวกสุดๆ เวลาเจอหน้าจอที่มีหลายส่วน

Toggle Switches (20 เมษายน)

ปุ่มเปิด/ปิดฟีเจอร์ต่างๆ ที่ Control UI ทำให้สลับการตั้งค่าเป็นเรื่องง่าย ไม่ต้องเปิดหน้า Settings บ่อยๆ

Gemini 2.0 Flash (16 เมษายน)

เพิ่มพลังแปลด้วยโมเดลใหม่ล่าสุด Gemini-2.0-Flash เร็วขึ้น คุณภาพดีขึ้นกว่าเดิม!

Hover-to-Force (16 เมษายน)

ฟีเจอร์ชี้ค้างเพื่อสั่งแปล ตอนนี้ใช้ได้กับทุกปุ่มที่เกี่ยวกับการ Force Translate

การปรับปรุงเพิ่มเติม

  • ปรับปรุง Control Panel ให้มีหน้าตาดูดีขึ้น ปุ่ม Preset กับส่วนควบคุม CPU ใช้งานง่ายกว่าเดิม
  • แก้บั๊กบันทึก Preset ที่ตั้งชื่อ Custom Preset แล้วบันทึกไม่ได้
  • ปรับปรุงหน้าตั้งค่า Model ให้มีหน้าตา UI/UX ดีขึ้น ปุ่ม Apply ชัดเจน
  • แก้ปัญหาการตรวจสอบ Unsaved Changes ในระบบ Preset
  • ปรับปรุงการป้องกัน Race Condition ใน Hover Translation
